# GTRX Landing Page

Marketing landing page for the GT-R marketplace/community built with Vite, React, TypeScript, and Tailwind (shadcn-ui).

## Development
- Install: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`

## Waitlist
Email capture uses Supabase. Set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in `.env` (see `src/lib/supabaseClient.ts` for defaults).***
